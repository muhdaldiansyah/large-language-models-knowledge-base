# Learning Activity Logs

## Log Format

Each log entry follows PHP-style single-line format:

```
[YYYY-MM-DD HH:MM:SS] [LEVEL] [CATEGORY] Message description
```

### Levels
- `INFO` - General information
- `LEARN` - Learning session activity
- `QUIZ` - Quiz/test attempts
- `REVIEW` - Review sessions
- `ERROR` - Issues encountered

### Categories
- `SETUP` - Repository/environment setup
- `CONTENT` - Content creation/updates
- `CLEANUP` - File cleanup/organization
- `SESSION` - Learning session start/end
- `TOPIC` - Topic studied
- `SKILL` - Skill used (tutor, quiz, etc.)
- `PROGRESS` - Progress milestones

## Log Files

- `learning.log` - Main activity log
